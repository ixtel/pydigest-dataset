Доступен релиз IPython 3.0, интерактивной оболочки для языка Python, которая сочетает возможности интерактивной консоли Python и командной оболочки Unix, предоставляет гибкие средства отладки, редактирования кода и визуализации данных. IPython активно используется в научной среде для разработки, обработки данных и интерактивного выполнения приложений, связанных с библиотеками numpy, matplotlib, sympy и scipy.

Новый выпуск примечателен появлением полноценных средств разработки на различных языках программирования, не ограничивающихся языком Python. Например, подготовлены модули для разработки на Bash, R, Ruby, Perl, Python 2, Python 3. Значительно переработан интерфейс web-окружения Notebook, позволяющего создавать документы, комбинирующие текст, исполняемый код, математические выражения, графики и мультимедийный контент. Реализована поддержка редактирования произвольных тестовых файлов. В web-интерфейс добавлен интерактивный эмулятор терминала.

Отмечается, что IPython 3.0 является последним выпуском, поставляемым в форме одного монолитного пакета. Начиная с версии 4.0 состав будет разделён на несколько пакетов: универсальные компоненты для работы с любыми языками программирования (notebook, qtconsole и т.п.) будут выделены в пакет Jupyter, а специфичные для языка Python части (интерактивный Python shell, Python kernel, IPython.parallel) продолжат поставляться под именем IPython.
