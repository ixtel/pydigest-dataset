# This project is unfinished/non-functional

## what.cd torrent downloader

what-dl uses Selenium WebDriver to ease the torrent downloading process on what.cd. Torrents are either downloaded locally or copied to ruTorrent, a web front-end for rTorrent. Users will be prompted to enter their what.cd (and possibly ruTorrent) credentials, afterwards just sit back and enjoy! Requires Firefox or Chrome browser to run (or a variant of either).

## Installation

pip install what-dl

or

pip install git+https://github.com/huntrar/what-dl.git#egg=what-dl

or

git clone https://github.com/huntrar/what-dl

cd what-dl

python setup.py install

## Usage

usage: what-dl.py [-h] [-a] [-al] [-aa] [-d] [-da] [-fl] [-l] [-m MAX_FILES]

[-mp] [-n] [-rt RUTORRENT] [-t] [-v]

[QUERY [QUERY ...]]

what.cd torrent downloader

positional arguments:

QUERY keywords to search

optional arguments:

-h, --help show this help message and exit

-a, --artists search for query in artists (default)

-al, --albums download albums only (default)

-aa, --aac download AAC audio files

-d, --dryrun run without downloading

-da, --download-all download albums and non-albums

-fl, --flac download FLAC audio files

-l, --local download torrent files locally (default)

-m MAX_FILES, --max-files MAX_FILES

maximum number of files to download

-mp, --mp3 download MP3 audio files

-n, --not restrict downloading chosen audio formats

-rt RUTORRENT, --rutorrent RUTORRENT

download using ruTorrent

-t, --torrents search for query in torrents

-v, --version display current version
