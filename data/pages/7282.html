Talk 1: Fast numerical optimization in Python

Abstract:

Python often has a reputation of being ill suited for heavy duty numerical computation. The traditional wisdom is to prototype in Python and rewrite in C++ when ready. This talk will showcase some very cool technologies such as Cython and Numba which allow programmers to minimize the amount of low level code needed while still achieving close to C++ performance and maintaining high level Python interfaces.

I will demonstrate these techniques by detailing implementations of numerical optimization algorithms for various machine learning tasks.  

Bio:  

Stephen Tu is a PhD student at UC Berkeley. 

Talk 2: Dask.array and Parallel PyData

Abstract:

Python has a tenuous relationship with parallelism due to the global interpreter lock (GIL). The PyData ecosystem (NumPy, Pandas, ...) can get around these issues by relying on its C/Fortran foundations. Dask is a library that leverages this workaround and parallelizes a broad subset of NumPy and Pandas operations.

This talk discusses the limitations of the GIL, the foundations of the PyData software stack, and the ongoing state of parallelizing Python data analysis by the PyData community.

Bio:

Matthew Rocklin is a computational scientist at Continuum Analytics and full time open source developer within the scientific Python ecosystem. He builds libraries for out-of-core and distributed computing that target non-expert users.
