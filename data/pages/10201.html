REPL (Read-eval-print loop) — среда для интерактивного программирования.

В такой среде пользователь может вводить выражения, которые среда тут же будет вычислять, а результат вычисления отображать пользователю. Название read-eval-print loop происходит от имён примитивов языка Lisp, которые реализуют подобную функциональность:

  • Функция read читает одно выражение и преобразует его в соответствующую структуру данных в памяти.
  • Функция eval принимает одну такую структуру данных и вычисляет соответствующее ей выражение.
  • Функция print принимает результат вычисления выражения и печатает его пользователю.

REPL-среда очень удобна при изучении нового языка, так как предоставляет пользователю быструю обратную связь. (все жадно взял с wiki)

Зачем нужен REPL? Очевидно что REPL нужен для обучения, проверки каких-то идей/кода, чтобы вспомнить как работает какой-то метод, для замены калькулятора, иными словаи когда цикл "Написал код -> запустил -> посмотрел результат" должен быть минимальным по времени.

Пару примеров таких циклов:

Пример: при парсинге страницы надо выдрать все блоки с текстом. Способы достижения разные - перебор различных css классов и id'шников, написание хитрого xpath или split или..

Пример 2: есть строка, необходимо выдрать из нее какие-то данные с помощью регулярного выражения. Редко когда получается регулярное выражение написать с 1 раза.

Чтобы запустить REPL для python достаточно в командной строке написать python:

$ python
Python 2.7.6 (default, Jun 22 2015, 17:58:13) 
[GCC 4.8.2] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

REPL поприветствует, скажет какая версия Python используется. Чтобы выйти из REPL можно нажать CTRL+C или написать exit()(или quit()) и нажать enter

Если Python у вас не установлен, можно REPL запустить в браузере - проектов много, например https://repl.it/languages/python3
