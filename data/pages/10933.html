Interesting Projects, Tools and Libraries

PEP 8 for Humans.
This document gives coding conventions for the Python code comprising the standard library in the main Python distribution.  This style guide evolves over time as additional conventions are identified and past conventions are rendered obsolete by changes in the language itself.

darksearch
A search engine for hidden material. Scraping dark web onions, irc logs, etc

Blinker
A fast Python in-process signal/event dispatching system.

Temporal.pw
Temporal.pw provides temporary secure storage for passwords so you can safely transmit them over insecure channels like E-Mail, Instant Messaging, SMS, etc.

shellpy
A tool for convenient shell scripting in Python. It allows you to write all your shell scripts in Python in a convenient way and in many cases replace Bash/Sh.

monica
monica is a command line chef that brings you tasty food.

hack.guides()
hack.guides() is an open-source CMS based on Markdown and Github written by-developers, for-developers. 

Octohook
Octohook is a server that listens for incoming webhooks, validates them and routes them to your code. The idea was to make it easier to write and deploy code that runs when something happens on your repo. Be it a new issue, a reverted commit or someone starring it.

do-latency
Digital Ocean latency checker helps to find fastest DO region from your location.

Pygemony
A tool written in Python to parse files for "TODO" and to raise an issue in the github repository with the todo message.
    
django-template-minimizer
Minimizes Django templates so that html is served up already minimized. Minimizes django templates and the html, in-line javascript, and in-line css inside.

conn-check
conn-check allows for checking connectivity with external services. You can write a config file that defines services that you need to have access to, and conn-check will check connectivity with each.
    
defopt
defopt is a lightweight, no-effort argument parser.

django-machina
Django-machina is a forum framework for Django providing a way to build community-driven websites. It offers a full-featured yet very extensible forum solution.

lifter
A lightweight query engine for Python iterables, inspired by Django ORM.

app-turbo
A web framework based on tornado for easier development, scaling up and maintenance.

speedcomplainer
A python app that will test your internet connection and then complain to your service provider (and log to a data store if you'd like).

Sidomo
If you can get your software to work in a docker container, then this module will make it work in Python.

braceless.vim
Text objects, folding, and more for Python and other indented languages.

lexiconjure
lexiconjure is a twitter bot that uses machine learning to define invented words, posting truncated definitions on Twitter and complete ones on Tumblr.
