rinoh-typeface-texgyreschola 0.1.0

TeX Gyre Schola typeface

This package provides the TeX Gyre Schola typeface for use with rinohtype. TeX Gyre Schola is based on URW Century Schoolbook L and can be used as a replacement for Century Schoolbook.
