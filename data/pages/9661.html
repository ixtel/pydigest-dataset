rinoh-typeface-texgyreheros 0.1.0

TeX Gyre Heros typeface

This package provides the TeX Gyre Heros typeface for use with rinohtype. TeX Gyre Heros is based on URW Nimbus Sans L and can be used as a replacement for Helvetica.
