django_loaddata_stdin 0.0.4

Django module to load data from stdin

Django loaddata from stdin extension

====================================

Installing

----------

Put it above the Django core in your installed apps.

INSTALLED_APPS = (

'django_loaddata_stdin',

'django...',

...

)

Usage

-----

./manage.py loaddata --format=yaml - << EOF

model: badger.badger

pk: 1

fields: {}

EOF
