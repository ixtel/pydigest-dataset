pydpc 0.1.3

Python package for Density Peak-based Clustering

Clustering by fast search and find of density peaks, designed by Alex Rodriguez and Alessandro Laio, is a density-peak-based clustering algorithm. The pydpc package aims to make this algorithm available for Python users.
