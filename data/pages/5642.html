pandashells 0.1.8

Command line data tools

Pandashells is a set of unix command-line tools that provide accessibility to the python data stack directly from the bash prompt. Users can utilize pandas-based data-munging, perform linear regression, and even take the spectral decomposition of non-uniformly sampled time series.
