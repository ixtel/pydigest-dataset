PyDev 3.5.0 is now available...

The main feature in this release relates to improvements to work with

py.test

(note: those that want to use it have to enable it in the PyUnit preferences -- besides having to install it properly in the used Python installation).

Mainly, the integration was changed so that the

py.test xdist

plugin can be properly used and module/session scoped fixtures should also work properly. The PyUnit UI in the preferences was also changed to show the preferences related to py.test when it's shown.

Still, there are other improvements: the

Model.objects

from django 1.7 is now available on code-completion again (as the static analyzer couldn't cope with it there are workarounds in the code), private attributes (those starting with

'__'

) are now shown when hovering over it on the debugger and there's a bunch of other bug-fixes and minor features available (see

http://pydev.org

for details).

Also, for those that don't know, there's an ongoing crowdfunding to keep PyDev support going on:

http://tiny.cc/pydev-2014
