pysimplelog 0.2.0

This is a pythonic simple yet complete system logger.

Package Documentation

This is a pythonic simple yet complete system logger. It allows logging simultaneously to two streams, the first one is the system standard output by default and the second one is designated to be set to a file. In addition, pysimplelog allows text colouring and attributes when the stream allows it.
