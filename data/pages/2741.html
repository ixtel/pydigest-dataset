About the benchmarks

The benchmarks show the relative speed of specific functions/algorithms when run in a variety of different libraries. The focus, to date, has been on comparing with NetworkX and igraph, two widely-used network analysis libraries.

All benchmarks were generated using the Benchmark class that is bundled with Zen. This class is a utility for quickly comparing the performance of different libraries, functions, or other arbitrary code paths.
