An evented server framework designed for building scalable and introspectable services and clients based on coroutines, built at PayPal.

Pinball
Pinball is a scalable workflow manager. It is designed to accommodate the needs of a wide range of data processing pipelines composed of jobs ranging from simple shell scripts to elaborate Hadoop workloads.

HoloViews
HoloViews is a Python library that makes analyzing and visualizing scientific or engineering data much simpler, more intuitive, and more reproducible.

Django: Wall of Superpowers!

Homu
Homu is a bot that integrates with GitHub and your favorite continuous integration service, such as Buildbot or Travis CI.

ChessM8
ChessM8 is a machine that can present an effective opponent in the game of chess. It moves the pieces around using a claw and a CoreXY system and detects the presence of the pieces with a reed sensor matrix underneath the board. The actual AI is any chess engine that uses UCI (almost all of them) and the machine connects to that via a Python script on a computer. Runs on an Arduino Mega.

TestTrack
