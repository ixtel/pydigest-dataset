CLITest

A command line interface functional testing framework

TestSuite

This is a class representing a collection of CLITest instances. In order to use this class, simply instanciate it and add CLITest instances to it using add_test(). Once all of the tests are added simply call run_tests().

CLITest

This is a class representing a test to be run against the cli. It is constructed with one parameter, command which is the cli command to test.

To use this class, first subclass it and add methods whose names start with "test" (for example "test_fails_when_not_passed_number" would be a good name). After that is done, create an instance of TestSuite and add an instance of your subclass to it using the add_test_method. Finally, run all of the tests in the test suite by calling run_tests. For example:

    from CLITest import CLITest, TestSuite
    from subprocess import CalledProcessError


    class TestEchoPrintsToScreen(CLITest):
        '''Tests whether the string passed in is the string
        passed out'''

        def test_output_contains_input(self):
            self.assertNotIsInstance(self.output, CalledProcessError)
            self.assertIn("test", self.output)

        def test_ouput_equals_input(self):
            self.assertNotIsInstance(self.output, CalledProcessError)
            self.assertEqual("test", self.output)

    suite = TestSuite()

    suite.add_test(TestEchoPrintsToScreen("echo test"))

    suite.run_tests()
