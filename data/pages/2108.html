Interesting Projects, Tools and Libraries

runcython
runcython lets you compile and run cython in one line. There's no need to muck around with distutils or intermediate files. Using cython the typical way would require creating 5 distinct files, accum.pyx, accum.c, accum.so, setup.py, and use.py. That's a lot of moving parts to keep track of. runcython keeps things simple so that you can just focus on writing fast code.

rapid.py
rapid.py is basically a python template designed for creating high-fidelity prototypes fast. The main purpose of rapid.py is to allow people to create prototypes without having to implement unused code in the final version of their product. You can create an API for your mobile app, or a responsive web version of your app or service. Put rapid.py in a production environment and store data with MongoDB for analyze it later.

cite2c
Citations in the IPython Notebook.

Combi
Combi is a Pythonic package for combinatorics. It  lets you explore spaces of permutations and combinations as if they were Python sequences, but without generating all the permutations/combinations in advance. It lets you specify a lot of special conditions on these spaces. It also provides a few more classes that might be useful in combinatorics programming.

Geeknote
Are you a geek? Do you like Evernote? Geeknote - is for you! Geeknote is an opensource Evernote console client for Linux, FreeBSD and Mac OS X. Use it for system administration needs, creating notes, notebooks, sync your local directories with Evernote notebooks.

GoogleScraper
A python module to search and scrape the mighty Google search engine using proxies (socks4/5, http proxy) and with many different IP's, including asynchronous networking support (very fast). 

CodernityREST
CodernityREST is a REST API framework made in Python. It aims to be configurable and fast. Designed to work with complex and big API, but will fit perfectly into small ones too. It takes care about data validation and error consistency (your and 3rd party).
