HabraFavorites

Статистика избранных статей с сайтов Хабрахабр, Geektimes, Megamozg в виде HTML-страницы или данных в различных форматах.

Установка

$ pip install habra-favorites

Приложение основано на фреймворке Scrapy. При возникновении ошибок при установке обратитесь к инструкции по установке Scrapy. Обратите внимание, что требуется Python 2.7, ветка 3.x не поддерживается.

Использование

$ habra_favorites [-h] [--version] [-d DOMAIN] [-f FORMAT] [-o NAME] username

Один обязательный параметр -- username. Логин пользователя, чьи избранные статьи будут собраны. Это может быть как Ваш логин, так и логин любого другого пользователя.

Опции:

  • DOMAIN -- сайт: habrahabr.ru (по умолчанию), geektimes.ru, megamozg.ru, all;
  • FORMAT -- формат результата: html (по умолчанию), json, csv, xml;
  • NAME -- имя файла для сохранения: по умолчанию favorites.html .

Результат

При нажатии на заголовки колонок созданной HTML-страницы посты отсортируются в соответствующем порядке. Повторное нажатие на активный заголовок пересортирует посты в обратном порядке.
