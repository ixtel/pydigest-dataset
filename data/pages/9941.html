Quick start

-----------

Install & settings

-------------------

1. Install the app

```

pip install django_iconpicker

```

2. Add "django_iconpicker" to your INSTALLED_APPS setting like this

```python

INSTALLED_APPS = [

...

'django_iconpicker',

]

```

3. Add Icon prefix in settings

```python

ICON_PREFIX = 'fa'

```

4. Run Test Server

```python

./manage.py runserver

```

Example usage

--------------

Example Models

--------------

```python

class Service(models.Model):

title = models.CharField(max_length=255)

icon = IconPickerField(max_length=255)

toto = IconPickerField(max_length=200, null=True, blank=True)

def __str__(self):

return self.title

```

Display

-------

First Load iconpicker_tags

--------------------------

Parameters : icon, title='', large=False, fixed=False, color=False

Example :

```Jinja

{% iconpicker_icon picker.icon 'My title' True '' '#FFF' %}

```

Include Font

------------

```Jinja

{% iconpicker_font %}

```
