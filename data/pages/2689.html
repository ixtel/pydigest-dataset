Sort of... however, you need a way of describing the pattern to match on and that may just be using strings.

Below is a really rough example of how it might be possible using some generic Actions and an ActionList, which can be queried using regular expressions... sort of.

import re

class Action(object):
  def __str__(self):
    return self.re

class Jump(Action):
  re = "J"
class Run(Action):
  re = "R"

class ActionList(list):
  def __init__(self, *args):
    super(ActionList, self).__init__(self)
    for i in args:
      self.append(i)
  def re_search(self,regex):
     s = "".join(str(i) for i in self)
     return re.search(regex,s)

al = ActionList(Jump(),Run(),Jump(),Jump())

Then calling:

type(al[al.re_search("JJ").pos])

gives the element at 3:

<class '__main__.Jump'>

While:

al.re_search("R.R")

Will give nothing.
