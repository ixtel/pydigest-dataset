Currency Portfolio Optimization Using ScienceOps
In this post, we'll take the view of someone (e.g. a bank or a mutual fund manager) with a lot of cash to invest in foreign currencies. We want to buy foreign currencies such that we maximize our return with respect to the US Dollar (USD) over the next month. We won't worry about the actual amounts invested. We simply want to maximize the percent return given that we convert 100% of some known amount into foreign currencies.

Web Scraping - It's Your Civic Duty
More and more information from local, state and federal governments is being placed on the web. However, a lot of the data is not presented in a way that is easy to download and manipulate. I think it is an important civic duty for us all to be aware of how government money is spent. Having the data in a more accessible format is a first step in that process. In this article, we will use BeautifulSoup to scrape some data from the Minnesota 2014 Capital Budget. Then load the data into a pandas DataFrame and create a simple plot showing where the money is going.

Learning Selenium Testing Tools with Python
The book's main aim is to cover the fundamentals related to Python Selenium testing. You will learn how the Selenium WebDriver Python API can be integrated with CI and Build tools to allow tests to be run while building applications. This book will guide you through using the Selenium WebDriver Python client library as well as other tools from the Selenium project. Towards the end of this book, you'll get to grips with Selenium Grid, which is used for running tests in parallel using nodes for cross-browser testing. It will also give you a basic overview of the concepts, while helping you improve your practical testing skills with Python and Selenium.

Interesting Projects, Tools and Libraries

Wifiphisher
Wifiphisher is a security tool that mounts fast automated phishing attacks against WPA networks in order to obtain the secret passphrase. It is a social engineering attack that unlike other methods it does not include any brute forcing. It is an easy way for obtaining WPA credentials.

Jam.py
Jam.py is an event-driven framework for developing database applications. The server side is written in Python and uses Web.py library, the client side in JavaScript and uses JQuery and Bootstrap

elasticsearch-gmail
Index your Gmail Inbox with Elasticsearch.

PyTeaser
PyTeaser takes any news article and extract a brief summary from it. Summaries are created by ranking sentences in a news article according to how relevant they are to the entire text. The top 5 sentences are used to form a "summary". 

pgcli
Pgcli is a command line interface for Postgres with auto-completion and syntax highlighting.

pomegranate
pomegranate is a package for graphical models and Bayesian statistics for Python, implemented in cython. It grew out of the YAHMM package, where many of the components used could be rearranged to do other cool things.

pygubu
Pygubu is a RAD tool to enable quick & easy development of user interfaces for the python tkinter module. The user interfaces designed are saved as XML, and by using the pygubu builder these can be loaded by applications dynamically as needed. 

sophies
Provides a simplified and more expressive way of exposing choices, in particular for use with Django models.
