slack-imgur

Get a random image from Imgur in a Slack Channel, with Slack Outgoing WebHooks.

Usage

Create a Slack Outgoing WebHooks, select a channel, set a trigger word and add slack-imgur.herokuapp.com to URL(s) field.

Call a random image

Call a random image from Imgur (if you set imgur as trigger word)

imgur

Tag filter

Get a random image tagged with the tag called lolcat

imgur lolcat

Run in your server

Clone the repository and run gunicorn

gunicorn app:app --log-file=- --reload -b address:port
