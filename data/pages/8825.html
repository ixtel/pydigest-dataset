pynano 0.1.1

A simple Python library for the NaNoWriMo API

This project provides a simple, Pythonic library to access the NaNoWriMo API:

>>> from pynano import User
>>> kromey = User('kromey')
>>> kromey.wordcount
64133
>>> kromey.name
'Kromey'

Objects are provided to access data about Users, Regions, and the Site itself. Be sure to read the docs to get started.
