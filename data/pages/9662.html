rinoh-typeface-texgyrepagella 0.1.0

TeX Gyre Pagella typeface

This package provides the TeX Gyre Pagella typeface for use with rinohtype. TeX Gyre Pagella is based on URW Palladio L and can be used as a replacement for Palatino.
