Usage

Just replace the variable username and password with your username (john.smith@gmail.com) and your password (password12345). You must use a gmail account, or you can modify the code to hook into the Yahoo! SMTP server if you want, shouldn't be very hard.. Also change FROMADDR and TOADDR to the corresponding email addresses. Then package it into a .exe or what ever if you want and when the user exits the game all of the keys typed during the time the game was open will be emailed to you
