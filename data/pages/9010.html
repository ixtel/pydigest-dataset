Aerate 0.0.1.dev24

Aerate REST APIs: falcons with bravado and swagger.

Aerate is an open source Python REST API framework built on top of falcon and based on swagger.io specifications parsed within bravado-core

Aerate can be used to build and deploy highly customizable, fully featured RESTful Web Services that conform to swagger specifications.
