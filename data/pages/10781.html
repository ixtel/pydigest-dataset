Overview

Django Server Metrics adds traffic and performance tracking to your Django application. It comes with a fully responsive web interface and charts/graphs powered by Chartist.js.

The motivation behind this project is to track real traffic and performance of the Django application. Ad-blockers have become popular and many block third-party analytics by default - such as Google Analytics.

Installation

Installation using pip:

$ pip install django-server-metrics

Update settings.py

# Add 'metrics' to INSTALLED_APPS
INSTALLED_APPS = [
    'metrics',
]

# Add 'MetricsMiddleware' to MIDDLEWARE_CLASSES
MIDDLEWARE_CLASSES = [
    'metrics.middleware.MetricsMiddleware',
]

Update urls.py

urlpatterns = [
    r('^metrics/', include('metrics.urls')),
]

Run database migrations

$ python manage.py migrate
