nexusformat: Python API to access NeXus data

This package provides a Python API to open, create, and manipulate NeXus data written in the HDF5 format. It also includes a server to allow files to be opened on a remote server across a network. The ‘nexusformat’ package provides the underlying API for NeXpy, which provides a GUI interface.

The latest development version is always available from NeXpy’s GitHub site.
