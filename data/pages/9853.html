Gajja: Fake objects for real tests.

The gajja library provides a system of Python test double classes for specific system objects:

  • Filesystem entries
  • Subprocesses

The Korean word 가짜 (gajja; IPA ˈkaːt͡ɕ̤a) means “fake thing”.
