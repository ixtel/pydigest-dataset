Hello World App

            from wsgiref.simple_server import make_server
from pyramid.config import Configurator
from pyramid.response import Response

def hello(request):
    return Response('Hello world!')

if __name__ == '__main__':
    config = Configurator()
    config.add_route('hello_world', '/')
    config.add_view(hello, route_name='hello_world')
    app = config.make_wsgi_app()
    server = make_server('0.0.0.0', 8080, app)
    server.serve_forever()

Easy To Start

Let's create a minimal web app in 5 minutes.

1. Install Python

Install the latest Python 2 or 3 from Python.org.

2. Install Pyramid

For example, after installing Python 3:

            $ pyvenv myproject
$ cd myproject
$ bin/pip install pyramid

3. Create your first app

Using your editor, put the Hello World App code into a new file named app.py.

4. Run your app

            $ bin/python app.py

5. Open your app

Visit the URL http://0.0.0.0:8080 in a web browser.
