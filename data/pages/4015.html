In this show we speak with Jesse Davis from MongoDB. Jesse is the maintainer for a number of popular open-source projects including the Python MongoDB driver known as PyMongo and Mongo C (for C/C++ developers, yes you read right! C developers).

Jesse discusses how interesting it is to write both Python and C code and how it reawakens part of the brain.

You'll learn a little about MongoDB, how it compares to RDBMSes as well as other NoSQL data stores.

Join Michael and Jesse for these great topics and much more!
