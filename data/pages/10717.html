CLI

To upload a file simply run:

uguu path\to\file\here.txt                #uploads the file and returns the url. Don't forget the file extension!

To give the file a custom name:

uguu -n custom-name-with-extension.txt path\to\file\here.txt

To give the file a random name:

uguu -r path\to\file\here.txt

Python Module

To upload a file simply run:

from uguuAPI import uploadfile

url = uploadfile("path\to\file\here.txt") #uploads the file and returns the url. Don't forget the file extension!
print url                                 #https://a.uguu.se/randomstring_filename.txt

To give the file a custom name:

from uguuAPI import uploadfile

url = uploadfile("path\to\file\here.txt", name="custom-name-with-extension.txt")
print url                                 #https://a.uguu.se/randomstring_customname.txt

To give the file a random name:

from uguuAPI import uploadfile

url = uploadfile("path\to\file\here.txt", randomname=True)
print url                                 #https://a.uguu.se/randomstring.txt
