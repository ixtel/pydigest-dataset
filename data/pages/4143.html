onradio.py Online Clock Radio script

This is a script i wrote for a clock radio, this is the derived version i created for the PC it doesn’t have all the functions that the raspberry pi version has. So what dose this script does , it gets data from the Ice Cast Radio database and allows to search and listen to the songs on your PC at the comfort of your terminal.

Requirements

  1. Linux / Macintosh System.
  2. Mplayer
  3. espeak (mac users can modify the script to use the “say” command)

Mplayer soft-volume fix

mplayer has a problem using its own volume control , in default Mplayer will use system volume which isn’t going to work for some functions in my applications this can be changed with this edit.

nano ~/.mplayer/config

add the line

softvol=1

Running the Program 

Clone The GIT and navigate two the source directory
git clone https://github.com/KaveenR/onradio.py
cd onradio.py/

run the program

python2 onradio.py

Basic Functions

>list genre 80's
> list stations 1133
>play 3753
>time
