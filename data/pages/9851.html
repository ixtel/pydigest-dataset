Custom Ansible Modules

This will be an instructor led walk-through on creating your own custom Ansible module. The example to be live coded (*fingers crossed*) will involve hitting a Magic 8-ball REST API to retrieve a response to a user-supplied question and will fail if the response is negative and continue if the response is affirmative.

Jonathan Davila is a Principal Architect at Red Hat (Ansible Team) where he specializes in DevOps consulting, Ansible AWS modules, and is a co-maintainer of the Ansible Lockdown project. Previously he was contracting for NASA HQ as DevOps Lead.

Computer Recreations

When teaching programming, it's difficult to come up with examples and exercises that are both not too complicated yet still interesting. This talk will revisit a historical source of interesting programming ideas: the 1980s Computer Recreations column in "Scientific American".

Andrew Kuchling is a longtime member of the Python community, and is currently a senior developer at Cox Media Group, helping maintain and improve the back-end infrastructure for Cox's content management system.
