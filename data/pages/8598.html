pixiv 0.1.1

Pixiv API client.

python-pixiv: Pixiv API client for moe girls.

Quickstart

Install python-pixiv:

$ pip install pixiv

Login to pixiv:

from pixiv import login

pixiv = login('username', 'password')

Save the work from a particular user:

user = pixiv.user(7631951)

for art in user.works():
    art.save()
