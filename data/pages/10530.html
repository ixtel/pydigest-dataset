django_shark 0.1.6

Django based bootstrap web framework

The Shark framework is a framework that allows for creating MVPs super fast. Django is great for creating models and views, but you still have to write your own html and css. In Shark there’s no need for this. You define in your view what you want to see and it gets rendered using Bootstrap and all html, css and javascript is generated for you.

Started docs and more info at http://www.getshark.org/
