Little functions that live in my pythonpath.

Installation

git clone http://github.com/dufferzafar/.python ~/.python
export PYTHONPATH=$PYTHONPATH:$HOME/.python

Test that it works:

$ python
>>> import pretty
>>> pretty.json(dict(this="should", be="in", cool="colors"))

To set the environment when you load your shell:

echo "export PYTHONPATH=$PYTHONPATH:$HOME/.python" >> .bashrc
echo "export PYTHONPATH=$PYTHONPATH:$HOME/.python" >> .zshrc

Learn more about PYTHONPATH.

Functions

browser.view_html

View an HTML string in a browser. I mostly use this along with the requests library to quickly view the response.content.

pretty.json

Pretty Print a dictionary/json in color. The idea came from jq.
