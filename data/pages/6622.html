download-files.py

This is a really small and simple utility I wrote for myself to download a given list of files. The list should reside in plain text, and each line should be one URL. You can also specify a directory for files to be downloaded to.

This utility is written for Python 3.x and has not been tested with Python 2.x.

Usage

Simply download and call:

python3 download-files.py

Without any arguments it will use the default urls.txt file in this repository, and download the files to <CURRENT_DIR>/downloads.

Arguments

  • -i (--input-file): The file containing your list of URLs.
  • -d (--download-dir): The directory you want files downloaded to.
