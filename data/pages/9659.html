rinoh-typeface-texgyrechorus 0.1.0

TeX Gyre Chorus typeface

This package provides the TeX Gyre Chorus typeface for use with rinohtype. TeX Gyre Chorus is based on URW Chancery L Medium Italic and can be used as a replacement for ITC Zapf Chancery(R).
