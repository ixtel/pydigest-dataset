Python Meetup октябрь: Deliberate Practice и десктоп-приложения на Penta.by

Всем привет!

Спешим поделиться видеозаписями выступлений с очередной встречи минского Python-сообщества.

Под катом вы найдете доклады:

  • Deliberate Practice: Coding Dojo, Code Kata and Coderetreat / Сергей Сергиенко

  • WRK: Modern HTTP benchmarking tool / Алексей Романов

Приятного просмотра!

Deliberate Practice: Coding Dojo, Code Kata and Coderetreat / Сергей Сергиенко

Сергей посвятил свое выступление правильному и эффективному развитию инженерных скиллов и мастерства разработки без отрыва от производства, а также привязки к определенной технологии или языку программирования.

Основной темой доклада стала «Осознанная практика» — как она может быть реализована при помощи Coding DoJo, Code Katas и Code Retreat.

Быстрая разработка десктоп-приложений с Penta.by / Андрей Пучко

Андрей рассказал об истории и архитектуре платформы для разработки десктоп-приложений Penta.by, а также показал основные этапы создания приложения.
