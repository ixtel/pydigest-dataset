Articles, Tutorials and Talks

An Algorithm to Extract Looping GIFs From Videos
Finding and extracting well-looping segments from a movie requires much attention and patience. To make things easier I wrote a Python script which automates the task. This post explains the math behind the algorithm and provides a few examples of use.

Naive Bayes Classifier using Python and Kyoto Cabinet
This post describes how to build a simple naive bayes classifier with Python and the Kyoto Cabinet key/value database. It begins with a short description of how a probabilistic classifier works, then  implements a simple classifier and put it to use by writing a spam detector. The training and test data comes from the Enron spam/ham corpora, which contains several thousand emails that have been pre-categorized as spam or ham.

Monte Carlo K-Means Clustering of Countries with Python
In the first part of this three-part series, What Drives Real GDP Growth?, I identified four themes which drive real GDP growth. These themes are based on 19 socioeconomic indicators whose average Spearman and Pearson correlations to real GDP growth were statistically significant and greater than 0.2 across the 188 country sample. In this article the 188 countries are clustered based on those 19 socioeconomic indicators using a Monte Carlo K-Means clustering algorithm implemented in Python. Clustering can help to reduce the amount of work required to identify attractive investment opportunities by grouping similar countries together and generalizing about them. 

Python for Forensics
Ashley A. Holtz covers typical forensic artifacts on Windows systems, several useful open source libraries, and techniques to analyze files. Ashley is a programmer at KPMG LLP in DC focusing on mainly security and forensics. 

Installing Django on IIS: A Step-by-Step Tutorial
This tutorial walks you through the basic setup of a Django project on IIS. It covers the installation of Python, Django, and related tools, including running the Django project both standalone and as a FastCGI server.

Interesting Projects, Tools and Libraries

vim-hackernews

djoser
REST implementation of Django authentication system. Djoser library provides a set of Django Rest Framework views to handle basic actions such as registration, login, logout, password reset and account activation. It works with custom user model.

django-watson
Full-text multi-table search application for Django. Easy to install and use, with good performance.

infernal-twin
This is the tool created to automate Evil Twin attack and capturing public and guest credentials of Access Point.

API-Hour
API-Hour is a lightweight daemon framework, that lets you write powerful applications. It was created to answer the need for a simple, robust, and super-fast server-side environment to build very efficient Daemons with ease.

dirs3arch
dirs3arch is a simple command line tool designed to brute force directories and files in websites.

django-offermaker
django-offermaker library is a solution for Django applications, which allows to create multi variant offers and use them on your site. Multi variant offer is a structure which defines dependencies between values for given set of fields.
    
ocropy

Toyz for Data Reduction and Analysis - Rochester, NY
Fred Moolekamp will talk about Toyz, a web framework based on Tornado that he's been working on to make analysis of large Astronomical data sets much more interactive. Some of the tools currently implemented in Toyz are an image viewer that can quickly display large (>1Gb) images on demand and interactive plots, which can be linked together to allow users to more easily view high dimensional datasets. Since Toyz is intended for scientific use, it is designed to integrate with custom code to allow users to easily create GUI controls and interactive tools.
