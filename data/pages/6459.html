Новые курсы программирования на языке Python от Фоксфорд

Курсы Python от Фоксфорда предназначены для тех, кто хочет изучить, начиная от азов и до профессиональных возможностей, современный, востребованный и в то же время очень понятный язык программирования.

Я уже писал ранее про данные курсы.

В этом учебном году Фоксфорд снова запускает курсы изучения Python:

  • Язык Python для начинающих (подходит для детей с 6 класса, а также для тех, кто только начинает изучение Python).
  • Язык Python (подходит для детей с 8 класса, для тех, кто хочет улучшить свои навыки программирования). Будет рассмотрена разработка игр, в том числе для Android.
  • Подготовка к олимпиадам по информатике (специально для старшеклассников, мечтающих научиться решать задачи на языке Python).
