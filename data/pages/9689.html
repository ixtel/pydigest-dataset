## Description

Optimizing Python applications without mutilation code

## Usage

### Decorator:

```python

from opyum import optimize

@optimize

def function_for_optimize():

...

```

### Import-hook:

```python

import opyum

opyum.activate()

# other imports

```

### Command-line mode:

$ opyum show myfile.py

$ opyum diff myfile.py

## Installation

Installation is simple with pip:

$ pip install opyum

or with setuptools:

$ easy_install opyum
