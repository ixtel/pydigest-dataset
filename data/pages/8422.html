A simple Django app to authorize

====

AUTH

====

auth is a simple application that allows certain users into

parts of you application depending on the groups that they

belong

Quick start

-----------

1. Add “auth_template” to your INSTALLED_APPS setting like this::

INSTALLED_APPS = [

...

‘auth_template’,

]

2. Include the polls URLconf in your project urls.py like this::

url(r'^auth/', include('auth_template.urls')),

3. Run the ‘python manage.py create_groups’ to create groups

4. In your project settings you must also have the following

variable so that the auth can redirect to the appropriate

page once a user is authorized

APPURL = 'your applications home/start page'

5. In your setting you must also have settings for email backend

6. To protect your class based views import method_decorator and

from auth_template.utils import group_required like so

from django.utils.decorators import method_decorator

from auth_template.utils import group_required

now decorate the dispatch method of your class like so

--------------------------------------------------------------------

class ReportBrowser(TemplateView):

@method_decorator(group_required('app_users'))

def dispatch(self, *args, **kwargs):

return super(ReportBrowser, self).dispatch(*args, **kwargs)

template_name = "index.html"
