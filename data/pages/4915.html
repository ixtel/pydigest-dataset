Getting Started with the Python SDK

Rackspace's official SDK for Python is pyrax, an Apache licensed SDK for OpenStack clouds.

Pyrax supports these Rackspace services directly

A unified OpenStack SDK for Python is in the works.

Requirements

Using pyrax requires Python 2.7.x. There are plans to port it to run in both 2.x and 3.x, but that work has not yet been started. We recommend installing pip to handle all of the dependency requirements for pyrax.

Installation

Generally, pyrax should be installed into a virtual environment using pip:

Getting started with Pyrax

import pyrax

pyrax.set_setting("identity_type", "rackspace")
pyrax.set_default_region('DFW')
pyrax.set_credentials('{username}', '{apiKey}')
