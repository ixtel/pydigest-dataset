imporganizer 1.0.6

A Python import organizer

Organizes import of Python files in alphabetic order.

Your imports should be at the start of the file with two empty lines separating them from the rest of the code. For long lines do not use parenthesis. Examples:

GOOD:

from django.models import Model, \
    Manager

BAD:

from django.models import (Model,
    Manager)
