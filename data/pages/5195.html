Sharpy

Overview

Sharpy, A Simple but robust app in PyQt4.

Why Sharpy?

Because I was using a SHARP calculator to test some functionality. Hence the name Sharpy.

Why would I use it?

I didn't really create this calculator for people to use. It doesn't have anything special.
I created this as my first PyQt project so I can have a feel of how things are done in PyQt and I can tell by now that I'm loving it.
Now the answer to this question is, I added some useful comments in the source code, so you might want to have a look at it if you are learning PyQt or maybe building an advanced calculator in it.

I didn't feel like creating a blog and writing a tutorial. I'd rather be coding than doing that, so I commented my code for anyone trying to dig in. Hopefully later I will build some more useful apps in PyQT.

How do I test it?

Oh, I thought you would never ask. What's your OS?

I am using MAC OS X

Good then, I created a sharpy.dmg file of this app so you don't have to install anything else. Just download this project and run 'sharpy.dmg' then copy the app into a folder or run it directly. Easy, right?

I am using XYZ OS

So you're not using a mac? Or you want to run this app without using the .dmg file on your mac? Alright, I like to live dangerously too ;), just install these on your computer first (Look it up on Google for how to install them on your specific system):

  • Python (Obviously)
  • Qt
  • PyQt4
  • sip

After installing these, simply run
python sharpy.py

I want to build the executable file myself

Good, that's why I also uploaded the 'setup.py' file. This file will work only for py2app (on Mac), if you're gonna build it on Windows/Linux, you'll have to modify it a bit (look it up on Google).
So here's how you build it on a Mac:
First you install the py2app cli with:
sudo easy_install -U py2app
Then you do:
python setup.py py2app
Then for the dmg file:
hdiutil create sharpy.dmg -srcfolder dist/sharpy.app
That's it! Really.
