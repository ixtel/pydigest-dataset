rinoh-typeface-texgyreadventor 0.1.0

TeX Gyre Adventor typeface

This package provides the TeX Gyre Adventor typeface for use with rinohtype. TeX Gyre Adventor is based on URW Gothic L and can be used as a replacement for ITC Avant Garde Gothic.
