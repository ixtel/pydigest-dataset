Pyvona is a simple Python wrapper for Amazon’s IVONA text-to-speech. With Pyvona, great sounding text-to-speech in Python is simpler than ever.

Installing Pyvona

To install the Pyvona library from PyPi, type pip install pyvona into a terminal with elevated privleges.
If you prefer doing things the hard way, please follow these steps:

  1. Download the latest version of Pyvona here.
  2. Unpack the archive and navigate into the resulting folder.
  3. Install the requests and pygame packages if you do not already have them on your system. Although pygame is not required for tts audio fetching, it is required for Pyvona’s speak method.
  4. Run the command python setup.py install  with elevated privileges.
  5. If you haven’t already visit here to set up an IVONA account and create credentials. Store these credentials in a safe place as they are needed to make requests to IVONA’s servers.

Pyvona Sample Code

The following shows a code example that uses a Pyvona Voice object to say “Hello World”. When using this code snippet, replace Access Key and Secret Key with the credentials you created in Step 5 of the Pyvona installation above.

import pyvona

v = pyvona.create_voice(‘Access Key’, ‘Secret Key’)

v.speak(‘Hello World’)

 Pyvona API

All of Pyvona’s functionality is contained within a Voice object. To create a Voice instance, use Pyvona’s create_voice method:

  • pyvona.create_voice(access_key, secret_key)
    •  Constructs and returns a Voice object

All interfacing with IVONA is completed through Pyvona’s Voice object:

  • class pyvona.Voice
    • The Voice object used to interface with IVONA’s tts API
    • codec
      • The codec with which to save voice responses. Options are ‘mp3’ and ‘ogg’.
    • paragraph_break
      • The paragraph time pause in ms. Default is ‘650’.
    • region
      • The AWS server cluster to contact. Options are ‘us-east’, ’us-west’, ’eu-west’. Default is ‘us-east’.
    • sentence_break
      • The sentence time pause in ms. Default is ‘400’.
    • speech_rate
      • The rate of text speech. Options are ‘x-slow’, ’slow’, ’medium’, ’fast’, ’x-fast’. Default is ‘medium’.
    • voice_name
      • The name of the desired speaker. To see all speaker options, use the list_voices() method
    • fetch_voice(text_to_speak, filename)
      • Saves the speech audio for a given to text to the given file name with the codec extension.
    • fetch_voice_ogg(text_to_speak, filename) DEPRICATED: see fetch_voice
      • Saves the speech response for a given text, text_to_speak, to the given filename with a .ogg extension.
    • list_voices()
      • returns the currently available IVONA voices in a JSON format. Useful for determining the desired voice_name to use.
    • speak(text_to_speak)
      • Speak the given text, text_to_speak, with the configured voice settings.

FAQ

Q: I just got new keys but keep getting the error: Error fetching voice: {“Message”:”Authentication failed”}. Is something wrong with my keys?

A: Ivona’s servers often take a few hours before they recognize new keys. Wait a few hours and, if the problem still persists, submit a comment below.

Q: When I try to speak some text, I keep getting a “click” sound. What’s wrong?

A: First, check for errors with your request. It may be the case that you are having authentication problems with your keys. If you are on OSX Yosemite, try typing the following (courtesy of sinanm89):

brew uninstall sdl_mixer

brew install sdl_mixer
