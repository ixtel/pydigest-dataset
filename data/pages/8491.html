conguide 0.9.4

Scripts to create a "Convention Guide" or "Pocket Program"

This package helps automate the creation of the Convention Guide (a.k.a. Pocket Program) for science fiction conventions. It takes a .csv or .xml file of raw schedule information, and reformats it in several ways for inclusion into an InDesign document.

For more information, see WORKFLOW.rst.
