asphalt-py4j 1.1.0

Py4J integration component for the Asphalt framework

This is a component for the Asphalt framework that provides Py4J integration.

Py4J allows you to call Java code from Python by either spawning a slave JVM (Java Virtual Machine) in a subprocess or by connecting to an existing JVM.
