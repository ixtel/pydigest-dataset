rinoh-typeface-texgyrecursor 0.1.0

TeX Gyre Cursor typeface

This package provides the TeX Gyre Cursor typeface for use with rinohtype. TeX Gyre Cursor is based on URW Nimbus Mono L and can be used as a replacement for Courier.
