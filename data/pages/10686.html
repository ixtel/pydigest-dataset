rfcs 1.0.1

rfcs is the complete command-line tool to search and view RFCs.

The complete command-line tool for searching and viewing RFCs. It provides a convenient interface for the API provided by IETF

rfcs provides four major subcommands:

  • rfcs search QUERY - Search for RFCs matching a particular query.
  • rfcs info RFC - Get information on a particular RFC.
  • rfcs text RFC - View the RFC’s text.
  • rfcs url RFC --format FMT - Get a url to view the RFC, in a format of {text, html, pdf, bibtex}.

Installation

pip install rfcs
