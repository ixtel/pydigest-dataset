Is this a hosted service?

No. Sendy is a self hosted application that runs on your web server. Pay once and it's yours, there's no recurring fee.

What are the requirements?

You need an Apache server running a Unix like operating system (eg. Linux) with PHP & MySQL support. Almost all hosting companies support them.

Is Sendy easy to setup?

Yes. Just follow our Get Started Guide to get up and running.

Will you update Sendy?

Yes of course! We're using Sendy as well and will continually make it better and better. Updates are free up till the next major version, eg. version 3.x, 4.x etc.

What's SES send rate and limit?

Once signed up for Amazon SES, submit the SES Sending Limits form to request for your desired daily sending limit. Eg. if you need to send 10k emails daily, request Amazon for a daily sending limit of 10k emails per day.

Can I install Sendy on multiple domains?

Sendy can be installed on one domain per license. To install Sendy on more domains, you need additional licenses. Sendy's licensed domain is just 'where you install Sendy on'. You can send emails belonging to any domain you want.

Are Sendy's files obfuscated?

Sendy's files are not obfuscated except for the installation and functions file. You can edit the source code for your own personal use without needing these two files.

What's with Sendy's low pricing?

Some say the cheapest option isn't always the best value. What they may have failed to understand is that Sendy is an app that you host yourself and that it uses Amazon's sophisticated email infrastructure built over the years to serve its own large-scale customer base. Brands like Comcast, Consumer Reports & Instra understood this and have started adopting Sendy.
