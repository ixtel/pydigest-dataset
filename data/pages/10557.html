# cr-cli :wrench: CLI tool for working with collective-reaction data

## Installation

` $ pip install cr `

## Usage

` $ cr --help $ cr dump-pad ./path/to/pad.gz $ cr dump-game ./path/to/game.gz `

## Development

To work on the code here, run

` $ python setup.py install `
