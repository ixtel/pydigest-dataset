Всем привет!
На просторах гитлаба наткнулся на небольшую библиотеку exrex, позволяющую генерировать данные по регулярным выражениям.
API крайне простой count - возвращет количество возможных вариаций parse - парсер регулярных выражений getone - возвращает псевдо-рандомное значение generate - возвражает генератор со всеми вариантами

Примеры использования

На всякий замечу, что у вас по некоторым примером вывод может быть другим


>>> print exrex.getone('[A-Z][a-z]{1,10} [A-Z][a-z]{1,10}') Boris Yeltsin
>>> print exrex.getone('\+[0-9]\([0-9]{3}\) [0-9]{3}-[0-9]{2}-[0-9]{2}') +5(777) 790-68-69
>>> print [r for r in exrex.generate('|'.join(str(i) for i in range(5)))] [u'0', u'1', u'2', u'3', u'4']
>>> exrex.count('[0-9]{1}') 10
>>> print exrex.parse('[0-9]{1,100}') [('max_repeat', (1, 100, [('in', [('range', (48, 57))])]))]
Есть также возможность запуска из консоли
>>> python -m exrex -r [0-9]{2} 31 Возвращает псевдо-рандомное число из 2 цифр
За более подробной информацией в ссылки.
