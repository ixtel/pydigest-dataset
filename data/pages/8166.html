TermOut 1.1

Drop in solution for progress bars and logging

TermOut

Terminal output libraries for Python 2.7 and higher.

Progress Bar

Dynamically scaling and easily customizable progress bar.

|###################################################################| (100%)

from TermOut import ProgressBar

import time
progress_bar = ProgressBar.ProgressBar(30, ratio=1, fill_char="#")
i = 0
while i <= 30:
    progress_bar.update(i)
    time.sleep(0.2)
    i += 1

Logging

Flexible colorful logging with caller detection and output to the appropriate std.

Logging Image

from TermOut import Logging

Logging.info("Hi!", color=Logging.UNDERLINE+Logging.BACKGROUND_GREEN+Logging.RED)
Logging.header("This is a header")
Logging.warning("This is a warning")    # > stderr
Logging.error("This is an error")       # > stderr
Logging.success("Great success!")

Installation

From source:

Clone the repo

git clone https://github.com/PhilipTrauner/TermOut

Change into repo directory

cd TermOut

Run setup.py

python3 setup.py install

From PyPI

pip3 install termout
