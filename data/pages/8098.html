Компания MediaTek Labs объявила о доступности платформы для разработчиков MediaTek LinkIt Smart 7688, построенной на однокристальной системе MediaTek MT7688, которая была представлена в июне прошлого года. Как утверждается, новая платформа, работающая под управлением OpenWrt Linux, предоставляет разработчиками широкие возможности программирования и позволяет сократить сроки проектирования устройства с модулями Wi-Fi и доступом в облачные сервисы. В качестве примеров названы IP-камеры, системы наблюдения, умные бытовые приборы и шлюзы Wi-Fi.

Платформа доступна в двух вариантах, различающихся аппаратной составляющей: LinkIt Smart 7688 и LinkIt Smart 7688 Duo. Оба они построены на SoC MediaTek MT7688AN, но второй дополнительно оснащен Arduino-совместимым микроконтроллером. В любом случае в оснащение входит 128 МБ ОЗУ, 32 МБ флэш-памяти и модуль Wi-Fi, а также периферийные интерфейсы.

По мнению MediaTek, разработчики OpenWrt смогут использовать OpenWrt SDK для создания приложений на языке C для устройств IoT, применяя для реализации дополнительной функциональности более 2000 готовых программных модулей. Разработчики приложений для мобильных платформ и web смогут интегрировать свои наработки в среду IoT, поскольку платформа поддерживает Node.js и Python. Наконец, разработчики, отдающие предпочтение Arduino, смогут использовать IDE Arduino для разработки приложений под встроенный в LinkIt Smart 7688 Duo микроконтроллер и управлять периферийными устройствами в реальном времени.

Набор LinkIt Smart 7688 стоит $12,9, LinkIt Smart 7688 Duo — $15,9.
Версия Wi-Fi драйвера с open-source исходным кодом находится в разработке и будет доступна на сайте MediaTek Labs в течение нескольких месяцев.
