Articles, Tutorials and Talks

Hacking Education with Python - Data Mining Coursera for Popular Courses
This tutorial combines coursera data together with social media data to assess the popularity of courses. To to this, it uses the Coursera API to retrieve the course catalogue, sharecount.com API to get social media metrics for each course, and python's pandas library to query and order the courses by popularity.

Writing Redis in Python with asyncio: Part 1
A series of articles that shows how to write redis in pure python using asyncio library. In part 1, we will look at getting a basic redis implementation up and running using asyncio and fakeredis. You will be able to run basic commands such as get, set, lpush, lrange, sadd, smembers, hmset, hget, etc.

Entering Kernel Namespaces with Python
When working with Docker containers usually the question arises on how to connect into a running container without starting an explicit SSH daemon (which is considered a bad idea). This leads us to Linux Kernel namespaces.

Discovery Engines: Network Analysis
Network analysis has become an indispensable toolkit as more and more relational data are made available. In this video, F. Bill Shi of Knowledge Lab reviews the state of the art of modern network analysis, and provides a hands-on tutorial on three problems that have found successful applications in the real world (one of which has made billions of dollars): how to rank nodes, how to identify meaningful clusters of nodes, and how to predict potential links in a network.

Analyzing the Web For the Price of a Sandwich
The Common Crawl is a great public resource. You can scan over huge portions of the web with some simple tools and a price of a sandwich. MRJob makes this super easy with Python. Give it a try!

Sliding Windows for Object Detection with Python and OpenCV
In last week's post we discovered how to construct an image pyramid. And in today's article we are going to extend that example and introduce the concept of a sliding window. Sliding windows play an integral role in object classification, as they allow us to localize exactly "where" in an image an object resides.

Interesting Projects, Tools and Libraries

PyTricks
Collection of less popular features and tricks for the Python programming language.

cron-metrics
Implementation of a monitoring system of interval cron tasks in Python. Useful for metrics tasks that need to run every n minutes.

Hypothesis
Hypothesis is a library for property based testing in Python. You write tests encoding some invariant that you believe should always be true for a variety of inputs and then Hypothesis tries to prove you wrong.

The Python 101 Screencast
A Kickstarter project to raise funds to turn a book, Python 101, into a series of videos. The book, and consequently the video series, is aimed at beginners and intermediate level programmers. This series will be based on Python 3.

redislite
Self contained Python interface to the Redis key-value store. It makes it possible to use redis without the need to install and configure a redis server.

django_twitter
This is a simple twitter clone app built with Stream's API. It shows you how you can use GetStream.io to built a site similar to Twitter.

Spectra
Spectra is a Python library that makes color math, color scales, and color-space conversion easy.

RedisPAPA 
RedisPAPA is a redis monitor by using flask, angular, socket.io. It uses redis info to monitor the redis usage.

di-py
Dependency injection library for python. This library is designed to be very lightweight and flexible as to allow its use in a variety of scenarios, including their use to aid with unit testing.
