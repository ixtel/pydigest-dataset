微信(WeChat) 公众平台第三方 Python SDK，实现了普通公众平台和企业号公众平台的解析消息、生成回复和主动调用等 API。

阅读文档：http://wechatpy.readthedocs.org/zh_CN/latest/

安装

推荐使用 pip 进行安装:

pip install wechatpy

升级版本：

pip install -U wechatpy

从 0.8.0 版本开始，wechatpy 消息加解密同时兼容 cryptography 和 PyCrypto, 优先使用 cryptography 库。因而不再强制依赖 PyCrypto 库。如需使用消息加解密（企业号平台必须），请自行安装 cryptography 或者 PyCrypto 库：

# 安装 cryptography
pip install cryptography>=0.8.2
# 或者安装 PyCrypto
pip install pycrypto>=2.6.1

Tips: Windows 用户请先安装 PyCrypto 的二进制包后再使用 pip 安装 wechatpy 。 PyCrypto Windows 的二进制包可以在这里下载。
