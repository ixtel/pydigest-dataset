rinoh-typeface-texgyretermes 0.1.0

TeX Gyre Termes typeface

This package provides the TeX Gyre Termes typeface for use with rinohtype. TeX Gyre Termes is based on URW Nimbus Roman No9 L and can be used as a replacement for Times (new) Roman.
