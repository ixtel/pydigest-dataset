bcloud-core 3.8.2.dev1

A fork of bcloud

bcloud-core is a fork of bcloud.

Advanced features such as beatiful Gtk GUI, secure keyring, intelligent file manager, localization were all trimmed. Only core api logic was kept intending to minimize requirement.

For daily usage, the upstream is highly recommended.

python3
