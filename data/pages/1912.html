Android applications can only be packaged for Android on a Linux machine with Python, Kivy, python-for-android, Adroid SDK and Android NDK. As the installation may result a bit tricky and also it is not possible in Windows or Mac then Kivy.org is offering a VirtualBox VM with python-for-android pre-installed. Going a bit farther from aprendekivy.es we offer also a 32-bit Ubuntu VirtualBox with buildozer and Android SDK and NDK pre-installed. With this virtual machine you can not only create applications for Android, but also for Linux, Windows, Mac and all the supported platforms.

You can start following the steps below:

  1. Download our zipped virtual machine image from here [please, leave a comment for a broken link] and unzip it. (Around 4 Gb). You will get a file named ‘AprendeKivy.ova’.
  2. Download the appropiate version of VirtualBox  for your OS and install it.
  3. Start VirtualBox, in ‘File’ menu click in ‘Import Appliance…’.
  4.  Select the file ‘AprendeKivy.ova’ and click ‘Next’.
  5. In the Appliance Import Wizard you should check ‘Reinitialize the MAC address of all network cards’ and optionally you can change some other options like the amount of RAM dedicated to the gest OS depending on the RAM available in your host. Finally click ‘Import’.
  6. When import is completed you can start your virtual machine and start working with Kivy.
  7. Notes: root password is ‘kivy’, username is ‘kivy’ and its password is also ‘kivy’. Don´t forget this info as will be needed to install new applications, change configurations (language for example as the virtual machine is in spanish but can be changed to english easily) and manage the OS.
