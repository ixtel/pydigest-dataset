BaryPy

BaryPy is a Python3 script that uses Numpy to find the barycenter of some bodies. Simply provide it a list of bodies as input, defined as:

X:Y:Z:MASS

such as, for example:

10:-20:9.9:100

You can put these in a file (such as the provided example.bodies). You call the program like this:

./barypy.py [options] infile [outfile]

So, for example:

./barypy.py example.bodies

will run the example system and print the output to the screen.

The -m option toggles machine readable output (the output of the run can be copy-pasted to a .bodies file for input) and the -v option enables verbose mode, including reporting on run times for the computation and reporting the number of loaded bodies.

The test_barypy.py program will generate a list of bodies, to test the speed of the barypy program. Usage: ./test_barypy number
