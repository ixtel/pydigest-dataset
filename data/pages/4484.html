You can update to Django 1.8 quite easily. You just need to install Django 1.8 and update the

requirements.txt

file! However, if you want to follow the recommendations of Django 1.8, you’ll have to make two more changes:

  • Update the urls.py file so that it doesn’t use patterns (see the boilerplate file)
  • In settings/base.py change the TEMPLATE_DIRS by TEMPLATES (see the boilerplate file)
