This is an image carousel I made using python Beautiful soup 4 and Rainmeter. Source code here:

https://github.com/Sunnydaze/Pixiv-Scraper

How it works: You must have Rainmeter 3.2.1 Final Release or higher.. and Python 2.7 with beautiful soup 4 Simply open the "Pixiv_1.0.rmskin" file. Then click the update icon. once it's complete, refresh the skin by right-clicking on the arrow icon and click "refresh" on the context menu.
