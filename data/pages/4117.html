Question

Given a set of strings, return the smallest subset that contains prefixes for every string. (source)

Approach

Some problems just cry out for a particular approach, and this is one of them. Whenever you hear the word "prefix" you should probably start thinking "trie," and this problem is no exception. I talked quite a lot about tries yesterday, and this post is going to build on top of that one, so please head over there first if you don't know what tries are.

If you remember from last time, finding the strings encoded in a trie that possess a common prefix involves finding the node for the last character in the prefix, and then doing a depth-first search from that node to every leaf on the trie. Here we want to do something similar, but instead of searching for every leaf, we want to find the first leaf, which will represent the shortest word.

class MPSTrie(Trie):
    
    def minimum_prefix_set(self):
        current = self.root
        results = []
        prefix = ''
        nodes = deque([(current, prefix)])
        while nodes:
            node, prefix = nodes.pop()
            if None in node.children:
                results.append(prefix)
            else:
                for child in node.children.values():
                    nodes.append((child, prefix + child.value))
        return results

Takeaways

Knowing the right data structure to use turns a difficult problem into a simple one. If for nothing else you should brush up on your algorithms and data structures.
