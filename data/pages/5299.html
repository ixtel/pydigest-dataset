Разработка мобильных приложений для смартфонов - горячая тема. Больше нет сомнений, что в экосистеме Android в этом смысле развилась бешеная активность. Однако, когда мы приступаем к разработке "реального" приложения для Android, начинает казаться, что имеется одна единственная возможность: программить на Java. Не стоит унывать - на Python тоже можно писать под Android.

Существует несколько проектов, которые помогут это сделать:

Kivy -

http://kivy.org/

- библиотека для быстрой разработки приложений для Android. Имеет приятный интерфей (GUI), а также поддерживает многие функции Android, например, multi-touch.

SL4A(Scripting Level for Android) - библиотека, которая позволяет создание и запуск скриптов, написанных на различных языках сценариев прямо на Android-устройствах. Эти сценарии имеют доступ ко многим API, доступном для нормального Java приложения, но с упрощенным интерфейсом.

QPython - это скриптовый движок, работающий на Android устройстве (телефон,планшет). Движок позволяет работать с функциями устройства, например, считывать qrcode.

Pygame Subset for Android -

http://pygame.renpy.org/

- подмножество Pygame, которое позволяет писать игры на Python для Android.
