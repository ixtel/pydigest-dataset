Introduction to Classes in Python

This presentation aims to give a basic introduction to the concept of classes in Python.  It assumes knowledge of very basic Python syntax and functions but is geared towards novices looking to learn more.  We'll answer questions including What is a class? What is an instance of a class? Why use classes?
