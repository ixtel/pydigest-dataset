Dgen

Dgen is a prototype of a pseudorandom dungeon generator. The algorithm used within the generator is min-conflicts with some procedural tweaks. The prototype comes with a simple GUI for creating and saving maps in postscript format.

Example images and screenshots can be found in the assets folder. There are examples of ASCII based maps and graphical based maps using Tkinter.

Usage

$ python dgen.py

or

$ chmod +x dgen.py && ./dgen.py

Tested on Python 2.6, and Python 2.7. Python 3 support was attempted but not tested. Use with caution! Tested on Windows 7, 8 and Linux. Pypy was also tested and appears to work as intended.

An OSX application can be created if you have py2app installed. You can install via:

$ easy_install py2app

To build an app run,

$ python setup.py py2app

Dependencies

python version 2.6 or greater, may work on Python 3; I'm hopeful.

tkinter already included in most Python installations

idlelib already included in most Python installations
