About WAbot

WAbot is a WhatsApp bot written using yowsup ,which users can messege WAbot different commands and queries to recive jokes, quotes etc.

How to Run

rename login.sample.json as login.json with the correct inforamtion filled out. Open a Terminal on the project Dir and run

python bot.py
