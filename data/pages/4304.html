Pagegen static site generator

Pagegen can handle regular sites and blogs, organizing and generating content from flat text files and directories. Aims to generate a fully featured site ready for CSS styling.

  • Pages are simple text files with reStructuredText markup
  • Unlimited hierarchical page structure with menu and bread crumbs
  • Site search
  • At generation time executable files output is used as content
  • Web friendly URLs, sitemap, RSS feed, tags and categories
  • Generated html files can be hosted on any web server (FTP deployment included)
  • Hooks for running custom scripts during generation (e.g. automate CSS/Javascript compression)
  • Python 2.7
