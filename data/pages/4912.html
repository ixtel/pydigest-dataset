6-ая конференция DevConf 2015

Приглашаем посетить DevConf - самую крупную конференцию года для профессиональных разработчиков, которая пройдет 19 июня в Москве и состоит из секций: Storage, Python, Ruby, PHP, Javascript и общей секции.

Программа секции Python

  • asyncio: обзор ситуации на сегодняшний день (Андрей Светлов)
  • Yet another object level permissions (Сергей Лихобабин)
  • Centrifuge - мгновенные сообщения для веб-приложений, эволюция проекта от Python к Go (Александр Емелин)
  • view как чистая функция от состояния базы данных (Беда Илья)
  • Мастер-класс по асинхронному сетевому программированию на Python (Андрей Светлов)
  • Django DB Mailer — простая и удобная батарейка, для отправки почтовых сообщений в ваших проектах на Python и не только (Руслан Аскаров)

Подробнее о докладах: http://devconf.ru/ru/schedule#python

А 20 июня пройдут эксклюзивные мастер-классы:

  • Разработка крупного масштабируемого web 2.0 проекта с нуля (соц.сеть на 100 млн пользователей)
  • Создание кроссплатформенной мобильной игры с нуля
  • Sphinx 3.0 - поиск и поисковые сервисы

И много других. Полный список мастер-классов: http://devconf.ru/ru/schedule/mk
