Flask API
flask
Flask API is an implementation of the same web browsable APIs that Django REST framework provides. It gives you properly content negotiated responses and smart request parsing. It is currently a work in progress, but the fundamentals are in place and you can already start building kick-ass browsable Web APIs with it. If you want to start using Flask API right now go ahead and do so, but be sure to follow the release notes of new versions carefully.
