Projects.

 

openstack-docker (github.com)

If you haven’t heard of docker already, it is a tool to automate the deployment of applications into self sufficient Linux containers. Openstack-docker allows you to deploy docker on Openstack.

 

PlexConnect (github.com)

This is neat. PlexConnect allows you to use Plex on an stock Apple TV.. Check out the README for more details.


origami (github.com)

Origami is a lightweight package that allows you to serialize objects into a binary format.  

 

E.V.E. (github.com)

If you have been envious of seeing Jarvis the computer in Iron Man or would like to interact with your computer like the one on Star Trek this project is right up your alley. Click through to check out all the awesome.

 

graph-explorer (github.com)

Graph-explorer is an enhanced dashboard for Graphite. Lots of good here, click through for very detailed documentation.
 

hooked (github.com)

A git hook plugin system built using python.
